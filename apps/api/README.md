# Aozora School System v3 API

NestJS backend for the v3 system.

Current stage:

- API skeleton only.
- No real database connection required yet.
- Prisma schema is a placeholder until the v3 dev PostgreSQL project is created.

Commands:

```bash
pnpm --dir apps/api dev
pnpm --dir apps/api dev:watch
pnpm --dir apps/api build
pnpm --dir apps/api test
```
